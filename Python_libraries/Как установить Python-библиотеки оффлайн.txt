Как установить библиотеки оффлайн через pip:
1) Установить pip, если не установлен
Проверка версии pip(делается в cmd, или в терминале среды разработки):
py -m pip --version

pip 24.2 from C:\Users\...\Python313\Lib\site-packages\pip (python 3.13)

2) Скачать ПРАВИЛЬНЫЙ файл с сайта библиотеки
например с сайта pypi.org, там есть почти все библиотеки нужные
Например мы скачиваем библиотеку Pillow
В "Загрузка файлов" ищем нужный нам wheel(.whl)
Я использую Python 3.13(64-bit), Windows
Среди всех файлов находим последнюю версию библиотеки для нашей системы. 
Это pillow-11.0.0-cp313-cp313-win_amd64.whl.
pillow-название
11.0.0 - версия библиотеки
cp313-cp313 - версия Python для которой ставится данная либа
win_amd64 - Windows(64-bit)

За "cp313-cp313" нужно следить

Скачиваем файл в папку с библиотеками (если нет папки, то можно самим создать 
в любом месте) (я держу такую папку в каждом проекте)

3) Открываем cmd или терминал среды разработки
Нужно попасть в папку с нужным нам wheel(.whl)
Для этого можно прописать в терминале cd 'нужный_путь_к_файлу'
нужный_путь_к_файлу - просто ctrl+c ---> ctrl+v из адресной строки
Про кавычки не зыбывать!
Далее пишем pip install pillow-11.0.0-cp313-cp313-win_amd64.whl

Далее получим сообщение, что все успешно(надеюсь)

Файл типа .whl можно на флешке носить
